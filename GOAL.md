# GOAL

## 컨셉

`secu-ai-agent`는 **여러 장비에 설치**되고, **중앙 서버와 통신**한다.

서버가 각 에이전트에 요청을 보내면, 에이전트는 온디바이스에서 검사를 수행하고 결과를 돌려준다.

## 제공 서비스 (요청 기반)

서버 → 에이전트 리퀘스트를 받으면 아래 서비스를 제공한다.

| 요청 | 설명 |
|------|------|
| Docker 검사 | 컨테이너/이미지/런타임 보안 점검 |
| K8s 검사 | 클러스터·워크로드 보안 점검 |
| 보안 리포팅 | 점검 결과 수집·리포트 |

## 관계도

```
                 ┌─────────────────────┐
                 │   Central Server    │
                 │  Request / Report   │
                 └──────────┬──────────┘
           Docker / K8s / 보안리포팅 요청
                            │
        ┌───────────────────┼───────────────────┐
        ▼                   ▼                   ▼
 ┌─────────────┐     ┌─────────────┐     ┌─────────────┐
 │ Agent Host A│     │ Agent Host B│     │ Agent Host C│
 │ on-device   │     │ on-device   │     │ on-device   │
 └──────┬──────┘     └──────┬──────┘     └──────┬──────┘
        └───────────────────┼───────────────────┘
                            ▼
                      결과 / 리포트
```
